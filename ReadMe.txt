ИМИТАЦИОННАЯ МОДЕЛЬ СИСТЕМЫ МАССОВОГО ОБСЛУЖИВАНИЯ (СМО)
========================================================

Вариант задания: 4

РАСШИФРОВКА ВАРИАНТА:
---------------------
- ИБ: Источники - бесконечные (ИБ)
- ИЗ2: Закон генерации - равномерный (ИЗ2)
- ПЗ1: Закон обслуживания - экспоненциальный (ПЗ1)
- Д1031: Дисциплина буферизации - заполнение по кольцу (Д1031)
- Д1004: Дисциплина отказа - вытеснение последней поступившей (Д1004)
- Д2П2: Дисциплина выбора прибора - по кольцу (Д2П2)
- D2Б4: Дисциплина выбора заявки из буфера - по приоритету источника, при равенстве - последняя поступившая (D2Б4)
- ОД1: Отображение динамики - календарь событий, буфер, текущее состояние (ОД1)
- (ОР1: Отображение результатов - сводная таблица (ОР1) - реализовано, но не выводится в пошаговом режиме)

ОПИСАНИЕ МОДЕЛИ:
-----------------
Модель представляет собой сервисный центр с тремя типами источников заявок (И1 - гарантийный/высший приоритет, И2 - корпоративный/средний, И3 - частный/низший), 
ограниченным буфером (ёмкость 5) и тремя приборами (мастерами). Реализованы дисциплины D1031 (заполнение буфера), D1004 (отказ), D2П2 (выбор прибора), D2Б4 (выбор заявки). 
Работает в пошаговом режиме (ОД1).

РЕЖИМ РАБОТЫ:
--------------
- Программа запускается в пошаговом режиме.
- После каждого события (генерация, завершение обслуживания) выводится текущее состояние системы.
- Пользователь нажимает Enter для продолжения.
- Возможен выход по Ctrl+C.
